## Multi-Vendor E-Commerce Platform

### 1. Super Admin Features
- **Vendor Management**
  - Add New Vendor
  - Update / Modify Vendor Details
  - Delete Vendor
  - Enable / Disable Vendor Account

- **Customer Management**
  - Add New Customer
  - Update / Modify Customer Details
  - Delete Customer
  - Enable / Disable Customer Account

- **Product Management**
  - View All Products
  - Approve / Reject Vendor Products
  - Edit Product Details
  - Delete Products
  - Enable / Disable Products

- **Query & Support Management**
  - Handle Customer Queries
  - Handle Vendor Queries
  - Respond to Support Tickets

---

### 2. Vendor Features
- Vendor Registration & Login
- Manage Profile & Store Details
- Product Management
  - Add New Products
  - Update Product Information
  - Delete Products
  - Manage Product Availability
- Customer Management
  - View Customer
- View Orders & Order Status
- Respond to Customer Queries
- [ADV][FUT] Coupon and Discount Management
  - specific product
  - fixed amount
  - percentage based
    - % base, min 1500 cart value
  - limitied number of user ( starting 100 user )
  - fix time based
  - specific user

---

### 3. Customer Features
- Customer Registration & Login
- Manage Profile
- Browse Products from Multiple Vendors
- Add to Cart & Checkout
- Place Orders & Track Order Status
- Submit Queries / Support Requests
